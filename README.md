# 10b: Next.js Fullstack Auth & Posts App

A full-stack web application built with **Next.js (App Router)** featuring secure authentication, posts, and comments system with a clean service-based architecture.

---

## 🚀 Tech Stack

- **Framework:** Next.js (App Router, v16)
- **Database:** MongoDB (Mongoose)
- **Authentication:** JWT (jose) + HTTP-only Cookies
- **Password Security:** argon2
- **State Management:** Zustand
- **Styling:** Tailwind CSS + shadcn/ui
- **Forms & Validation:** React Hook Form + Zod
- **API Handling:** Axios
- **Utilities:** cookie, http-status-codes
